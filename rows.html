Table Rows

Each table row starts with a <tr> and end with a </tr> tag.

tr stands for table row.

Example

<table>
 <tr>
  <We are</td>
  <td>cells in</td>
  <td>one row.</td>
 </tr>
</table>

Displays:

We are cells in one row.
