Headers

A table just cannot exist without a header. A header shows what is displayed in a particular table. So how do you do it?

<table>
 <tbody>
  <tr>
   <th>Name</th>
   <th>Age</th>
   <th>Height</th>
  </tr>
  <tr>
   <td>Tom</td>
   <td>16</td>
   <td>185cm</td>
  </tr>
  <tr>
   <td>Ryan</td>
   <td>18</td>
   <td>190cm</td>
  </tr>
 </tbody>
</table>

Define the header with a <th>. It will set a cell to a header, which will make the text bold.

The table will look like this:

Name Age Height
Tom 16 185cm
Ryan 18 190cm
